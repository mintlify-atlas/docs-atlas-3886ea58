## [2.0.0] - 2026-02-22

<Tabs>
  <Tab title="Added">
    ### New Features
    
    - **Version Management**: Bump package versions to 2.0.0 across all packages
    - **Proactive Messaging**: Agent can now initiate conversations based on context and user patterns
    - **Update Checker**: Built-in update notification system to keep users on the latest version
    - **Landing Page**: New official landing page built with Svelte + Vite
    - **CI Overhaul**: Complete GitHub Actions workflow improvements (#10)
  </Tab>
  
  <Tab title="Changed">
